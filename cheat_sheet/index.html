Metadata

Tag Description
<!DOCTYPE html> Document type declaration for the web browser to render an HTML file properly. Must be the first tag in an HTML document.
<head> Encloses metadata such as the title and style sheets.
<html> Encloses the entire HTML file.
<link> Includes links for CSS stylesheets and fonts for the HTML file.
<title> The title of the web page, shown in the browser tab.

Sectioning

Tag Description
<article> Self-contained document such as a post, news article, or a blog.
<aside> A portion of the document that is somewhat related to the main content, such as a fun fact.
<body> Encloses the main content of the HTML file. Such as this table.
<footer> Footer info containing info on author, copyright, or links. usually at the bottom.
<header> Contains introductory information like navigation, logo, search form, and author name.
<h1> <h2> <h3> <h4> <h5> <h6> Section headings ranging from the largest (<h1>) to smallest (<h6>).
<main> Contains the main content of a document.
<nav> Represents the navigation section of the page, usually at the top or bottom.
<section> Encloses a section of a document. Not very specific and should almost always have a heading.

Text

Tag Description
<div> Generic container.
<figcaption> Caption for the parent <figure> element.
<figure> Self-contained content with the option for using <figcaption> for a caption as a child element.
<li> An item in a list contained in an ordered list (<ol>), or an unordered list (<ul>).
<ol> An ordered list represented typically with numbers.
<p> Paragraph element represented as blocks of text.
<ul> An unordered list represented typically with bullet points.

Inline Text

Tag Description
<a> Uses a href attribute to create a hyperlink to a new webpage, or a section on the curent page.
<b> Makes the text bold. However the proper thing to use is font-weight in CSS, or use the <strong> element.
<br> Cause a line
break in the text.
<em> Cause the text to be in italics.
<span> Generic inline element most commonly used to include class or id attributes for CSS styling purposes.
<strong> Like <b>, makes the text bold.

Media

Tag Description
<audio> An audio file embedded into the document using the src attribute.
<embed> Embeds an media into the document. Other tags are more specific and therefor should be used instead.
<img> Embeds an image into the document.
<video> Embeds a media player for video. use the controls attribute for common video controls.

Tables

Tag Description
<table> The parent element that encloses a table.
<tbody> The row that contain the main content of a table.
<td> Table data, contains information for one column in a row.
<tfoot> The footer for a table.
<th> Indicates the information in the row as a header.
<thead> Encloses the table's header.
<tr> Creates a row for the table.
